STRUMENTI LINEA

Gli strumenti "linea" sono definiti in file di testo.

Due strumenti linea, "utente" e "parete", sono integrati nei programmi. Gli altri vengono caricati dalla sottodirectory symbol/line di TopoDroid. Questa directory e' popolata con gli strumenti di linea presenti nell'apk di distribuzione. E' possibile eliminare gli strumenti non necessari, modificare quelli che si desidera modificare e aggiungerne di nuovi che non sono forniti nell'apk.

Il file dello strumento linea contiene linee che specificano la proprieta' dello strumento di disegno.

  • "symbol line" [obbligatorio]
  • "nome NOME", dove NOME e' il nome predefinito dell'area [obbligatorio]
  • nomi tradotti, ad esempio "nome-es ..." [opzionale]
  • "th_name NAME", dove NOME e' il nome di Therion e deve essere uguale al nome file [obbligatorio]
  • "group GROUP", specifica il gruppo a cui appartiene questa riga [opzionale, default nessun gruppo]
  • "color rgb alpha" (rgb e alpha in formato esadecimale) [rgb obbligatorio, alpha e' facoltativo]
  • "width SIZE" [opzionale, multiplo di "larghezza della linea di base"]
  • "dash PATTERN" [dash pattern opzionale]
  • "style STYLE" dove STYLE puo' essere "straight" o "xN"
  • "effect" inizia la definizione della linea ed e' seguito dai comandi path.
  • "endeffect" chiude la definizione della linea.
  • "csurvey LAYER TYPE CATEGORY PEN" (Livello Tipo Categoria Penna)
  • "endsymbol"
Gruppo
I tipi di linea nello stesso gruppo possono essere uniti insieme da "continuazione".

Questo e' usato per specificare che i vari tipi di linee "a parete" possono essere uniti insieme.

Percorso dell'effetto
Questa opzione e' usata per definire linee con uno stile complesso.
