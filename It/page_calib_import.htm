Importazione calibrazione

Questa finestra di dialogo si apre toccando Importa nella finestra di dialogo dell'elenco Calibrazione.

Viene mostrato l'elenco dei file CSV di calibrazione.

Tocca la voce che desideri importare.

L'importazione del file non riesce se non contiene una calibrazione, o se la calibrazione è già presente.

Esportazione CSV calibrazione Il formato di esportazione CSV di calibrazione contiene i dettagli della calibrazione (nome, data, indirizzo BT del dispositivo, commento, algoritmo), seguiti dai dati di calibrazione.

Ogni riga di dati contiene:

  • indice dei dati
  • Valori G e M (sei numeri interi)
  • gruppo
  • azimut, inclinazione e rotazione
  • Errore
  • Stato
