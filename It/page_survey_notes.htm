Note del rilievo

Questa finestra di dialogo si apre toccando il pulsante Note della finestra Lista battute, la finestra Informazioni rilievo e la finestra Disegno (in modalita' "spostamento").

Con questa finestra e' possibile scrivere e modificare un testo generico con note che integrano i dati e il disegno del tuo rilievo.

  • Salva: salva le modifiche e chiude la finestra
  • Indietro: chiude la finestra senza salvare le modifiche

Le note vengono salvate in un file di testo nella sottodirectory "txt" di "TopoDroid".
Puoi anche modificare questo file con altri editor di testo.
