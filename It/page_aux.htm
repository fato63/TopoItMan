App ausiliarie

Quando lavori con TopoDroid, potresti trovare utili alcune app complementari e ausiliarie:
  • Cave3D e' un visualizzatore 3D per i file Therion ed e' utilizzato da TopoDroid per la visualizzazione 3D
  • Proj4 e' un'app (molto semplice) di conversione di coordinate, utilizzata da TopoDroid per convertire i valori di longitudine/latitudine in altri sistemi di coordinate
  • ThManager e' una app per mettere insieme rilievi, esportati in formato Therion, in un progetto unico. Produce un file di configurazione thconfig.
  • MobileTopographere' un'app per il posizionamento. Puoi usarla per prendere le coordinate del punto e fare in modo che TopoDroid le legga dal file dei punti della app MobileTopographer.
  • una app per fare fotografie (di solito c'e' un'app Camera)
  • SQLiteViewer (o qualsiasi altra app SQLite-manager) puo' essere utile per ispezionare il database di TopoDroid
  • un File Manager (p.es.:FileManager+) e' sicuramente utile per gestire i file (ed eventualemnte fare copie di backup)
  • un editor di testo (p.es.:Jota Text Editor), puo' essere usato per vedere i file di testo

Inoltre, a seconda di come elabori i rilievi, potresti voler installare:

  • un editor/visualizzatore di immagini (normalmente Gallery e' la app preistallata)
  • un editor/visualizzatore di file DXF (p.es.: TurboViewer)
  • un visualizzatore di fogli di calcolo (p.es.: Google Sheets)
  • un visualizzatore di file SVG (p.es.: SVG viewer)
  • un gestore di file ZIP (p.es.: ZArchiver)
  • GoogleEarth
  • una app per eseguire xtherion (p.es.: AndroWish)
  • un visualizzatore di file di traccia (p.es.: Androzic)

Infine, l'app wakeup-touch risulta molto comoda durante la sessione di rilievo nella grotta. Per risparmiare la batteria, potresti voler impostare il tempo di spegnimento del display molto breve (per esempio mezzo minuto). Con un'app wakeup-touch non e' necessario premere il pulsante di accensione per riattivare l'android, ma e' sufficiente toccare il sensore di prossimita'. Io tengo il mio dispositivo in una scatola protettiva aprendo solo il coperchio. Il pulsante di accensione sul lato non e' quindi accessibile quando il dispositivo e' nella scatola e l'app wakeup-touch in questo caso risulta davvero utile.
