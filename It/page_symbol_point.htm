STRUMENTI PUNTO

Gli strumenti "punto" sono definiti in file di testo.

Due strumenti punto, "utente" e "etichetta" (testo), sono integrati nei programmi. Gli altri vengono caricati dalla sottodirectory symbol/point di TopoDroid. Questa directory è popolata con gli strumenti punto previsti nell'apk di distribuzione. È possibile eliminare gli strumenti non necessari, modificare quelli che si desidera modificare e aggiungerne di nuovi che non sono forniti nell'apk.

Il file dello strumento punto contiene linee che specificano la proprietà dello strumento di disegno.

  • "symbol point" [mandatory]
  • "name NOME", dove NOME è il nome predefinito dell'area [obbligatorio]
  • nomi tradotti, ad esempio "nome-es ..." [opzionale]
  • "th_name NOME", dove NOME è il nome di Therion e deve essere uguale al nome file [obbligatorio]
  • "color rgb" (rgb in formato hex) [obbligatorio]
  • "orientation FLAG" [facoltativo, falso se mancante, vero se FLAG è "1" o "si"]
  • "has_text FLAG" [facoltativo, falso se mancante, vero se FLAG è "1" o "si"]
  • "csurvey LAYER TYPE CATEGORY" (Livello Tipo Categoria)
  • "path" inizio della definizione del punto. Questo è specificato con i seguenti comandi:
    • "moveTo X Y"
    • "lineTo X Y"
    • "cubicTo X1 Y1 X2 Y2 X Y"
    • "addCircle X Y R": (X,Y) center, R radius
    • "arcTo X0 Y0 X1 Y1 FROM_ANGLE SPAN_ANGLE": (X0,Y0 - X1,Y1) rettangolo di delimitazione, angoli in gradi, 0 lungo asse X, la direzione è CW
  • "endpath" segna la fine della definizione dello strumento
  • "endsymbol"
Orientamento
Se l'attributo "orientamento" è impostato, il simbolo del punto può essere ruotato.
