La mappa finale

Questa sezione descrive l'uso di Therion o cSurvey per redigere la mappa finale.

Troverai, tra le altre cose, le note del rilievo (un file di testo), i file del disegno (file Therion th2) e il file Therion del rilievo. In questo file l'inclusione dei disegni viene inserita ma commentata. Per ottenere il disegno nei file PDF esportati e' necessario decommentarli. E' una buona idea aggiungere una mappa per ogni disegno (scrap); effettivamente due mappe una per il piano e l'altra per il profilo.

Se vuoi compilare il progetto Therion devi fornire un file thconfig che puo' essere semplice come
   source mio_rilievo.th
   export map -proj plan -o Pianta_grotta.pdf
   export map -proj extended -o Sezione_grotta.pdf

   In alternativa puoi lavorare con cSurvey. Apri il file di disegno esportato con csx con cSurvey: avrai la scheda dati compilata con i dati di rilievo ed i disegni correttamente allineati nell'area di disegno di cSurvey.
