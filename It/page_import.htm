Importazione rilievo

Questa finestra di dialogo si apre toccando il pulsante Importa nella finestra principale.

Visualizza l'elenco dei file che possono essere importati nell'app.

TopoDroid puo' importare i file del sondaggio nei seguenti formati:

  • Therion: ".th" file di dati, con formato dati semplice ("normale da-a-lunghezza-bussola-inclin." e opzionalmente LRUD). Gli scarti di Therion non vengono importati.
  • Compass ".dat" file (non completamente testato)
  • VisualTopo ".tro" file (non completamente testato)
  • PocketTopo ".top" file. Dati e disegni vengono importati (non completamente testati)
  • Archivi TopoDroid ZIP. Questi sono file zip speciali con le informazioni per ripopolare il database, i disegni (file ".tdr"), il file di annotazione e le foto.

Gli archivi zip sono cercati nella directory "zip" (che e' dove TopoDroid li salva). I file del rilievo vengono cercati nella directory "import". Pertanto e' necessario copiare il file del rilievo che si desidera importare in questa directory. Se il nome del rilievo del file da importare esiste gia' nel database, il file non viene importato.
