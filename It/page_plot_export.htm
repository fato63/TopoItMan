Esportazione disegno
I disegni sono archiviati in formato binario TopoDroid e possono essere esportati come:
  • Therion th2 file
  • cSurvey file
  • DXF 2D
  • SVG
  • File immagine PNG

Oltre all'esportazione Therion, c'e' un'opzione per salvare automaticamente un disegno in formato Therion ogni volta che viene chiuso.

Il file di esportazione cSurvey contiene sia la pianta che i disegni della sezione. Tuttavia non ha le sezioni-x. Le sezioni-x non possono essere esportate come file cSurvey e devi esportarle come immagini se vuoi usarle per la postelaborazione con cSurvey.

La risoluzione delle immagini PNG e' un'impostazione regolabile. Tuttavia un'immagine piu' grande e piu' dettagliata richiede piu' tempo della CPU e potrebbe non riuscire a causa dei limiti di memoria. Se non riesce, TopoDroid prova a ridurre la risoluzione finche' non riesce ad esportare lo schizzo come file immagine PNG.

I disegni possono essere esportati come file AutoCAD 2D DXF. La versione di AutoCAD dei file DXF esportati e' R12. L'esportazione DXF e' stata testata con VariCAD 2016-1-07 e BricsCAD 3.1.17. Non e' garantito che possa essere aperto da AutoCAD.
