Importazione posizione

Questa finestra di dialogo si apre toccando il pulsante Importa della finestra Posizione.

Con questa finestra puoi importare in TopoDroid le coordinate di un punto da un file di elenco punti Mobile Topographer.

  1. Apri Mobile Topographer e trova le coordinate del punto (icona "bandiera" sulla sinistra).
  2. Una volta raggiunta una precisione sufficiente nelle coordinate, salva il punto (terza icona "doc con piu'").
  3. Salva l'elenco (quarta icona "dischetto"). Il file viene salvato in "MobileTopographer/pointlist"
  4. Importa le coordinate in TopoDroid. I punti salvati sono mostrati in una lista; tocca il punto da importare.
Facoltativamente, puoi accedere all'elenco dei punti salvati in MobileTopographer (quarta icona "doc") e sposta il punto in alto con le frecce verdi.
