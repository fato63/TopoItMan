Impostazioni TopoDroid

IMPOSTAZIONI GENERALI
  • Directory di lavoro. Predefinito "vuoto" (cioè "TopoDroid")
  • Dimensione del testo: dimensione del testo negli elenchi. >=1. Predefinito 14
  • Dimensioni dei pulsanti {piccolo, normale, medio, grande, enorme}. Predefinito normale
  • Livello delle funzionalità {base, normale, avanzato, esperto}. Predefinito normale
  • Tastiera. Abilita tastiere personalizzate anziché la tastiera predefinita Android.
  • Co-rilievo. Predefinito OFF
  • Lingua {default, en, fr, es, it, ru, de, pg, slo, bg, cn, ukr, hu}. Default "default" (cioè locale)

IMPORT/EXPORT SETTINGS

  • PocketTopo color map Map PocketTopo colours to TopoDroid objecs
  • Estimate LR extend on import of Compass and Visual Topo files, Default YES
  • Default data export type {th,svx,dat,srv,top,tro,dxf,svg,csv} default th
  • Default scetch export type {none,th,csx,dxf,svg,png} default th
  • Therion maps first: export therion maps before survey data. default OFF
  • Therion stations: Add station points to Therion scraps. Default YES
  • Therion splays: Add splay lines to Therion scraps. Default YES
  • Compass station prefix. Default NO
  • Compass LRUD splay angle. LRUD minimum splay-leg angle.
  • Compass swap LR. Swap L and R on Compass export.
  • Survex/Topo line termination {Linux,Windows}. Default Linux
  • Survex splays TO station. Default NO
  • Survex LRUD. Add LRUD to Survex export. Default NO
  • SVG grid. Add grid to SVG export. Default OFF
  • SVG in HTML. Export SVG as HTML page. Default OFF
  • KML stations. Add stations to KML export Default ON
  • KML splays. Add splays to KML export Default OFF
  • PNG resolution [0.5, 10]. Default 1.5 [x200 px]
  • PNG background color (RGB in 0-100). Default "0 0 0" [black]
  • DXF sketch scale. Default 1
  • DXF splay endpoints. Default ON

IMPOSTAZIONI IMPORTAZIONE/ESPORTAZIONE

  • Mappa colori PocketTopo Mappa i colori PocketTopo agli oggetti TopoDroid
  • Stima LR extend nell'importazione dei file Compass e Visual Topo, Default YES
  • Tipo di esportazione dati predefinito {th, svx, dat, srv, top, tro, dxf, svg, csv} default th
  • Tipo di esportazione disegno predefinito {none, th, csx, dxf, svg, png} default th
  • Prima le mappe Therion: esporta le mappe therion prima dei dati del rilievo. predefinito OFF
  • Stazioni Therion: aggiungi punti stazione agli scrap Therion. Predefinito SÌ
  • Splay Therion: aggiungi le linee di splay agli scrap di Therion. Predefinito SÌ
  • Prefisso del caposaldo in Compass. Predefinito NO
  • Angolo di splay di LRUD. Angolo minimo del tiro perche' venga assunto come LRUD
  • Scambia LR in Compass. Scambia L e R nell'esportazione Compass.
  • Terminazione di linea Survex/Topo {Linux, Windows}. Linux predefinito
  • Survex splay caposaldo A. Predefinito NO
  • Survex LRUD. Aggiungi l'esportazione di LRUD a Survex. Predefinito NO
  • Griglia SVG. Aggiungi griglia all'esportazione SVG. Predefinito OFF
  • SVG in HTML. Esporta SVG come pagina HTML. Predefinito OFF
  • Capisaldi KML. Aggiungi i capisaldi all'esportazione KML. Predefinito ON
  • Splay KML. Aggiungi splay all'esportazione KML Predefinito OFF
  • Risoluzione PNG [0.5, 10]. Default 1.5 [x200 px]
  • Colore di sfondo PNG (RGB in 0-100). Predefinito "0 0 0" [nero]
  • Scala disegno DXF. Predefinito 1
  • DXF splay endpoint. Predefinito ON
    • Versione di AutoCAD {6,12}. Predefinito 6

DATI DEL RILIEVO

  • Squadra. Squadra di rilievo predefinita. Predefinito "vuoto"
  • Politica capisaldi: criteri di denominazione delle stazioni di rilevamento {no, splay+battuta diretta, splay+battuta inversa, battuta diretta+splay, battuta inversa+splay, battuta inversa+splay+battuta diretta, treppiede}. Predefinito "splay+battuta diretta".
  • Nomi capisaldi {alfanumerico, numerico}. Alfanumerico predefinito
  • Nome del caposaldo iniziale. Predefinito "vuotot"
  • Dimensioni miniature foto [80, 400]. Predefinito 200 pxl
OPZIONI UNITA'
  • Unità lunghezze {m, ft}. Predefinito m
  • Unità angoli {gradi sessagesimali, gradi centesimali}. Valore predefinito gradi sessagesimali
  • Unità di misura disegno {m, yd, 2ft}. Predefinito m
OPZIONI POLIGONALE
  • Tolleranza battute. Tolleranza tra i tiri della stessa battuta >0 [percento]. Default 0.05 (5%)
  • Massima lunghezza tiro. Predefinito 50
  • Minima lunghezza battuta. Predefinito 0.5
  • Tiri minimi. Numero minimo di tiri per battuta >=2. Predefinito 2
  • Angolo vert. extend. Limite definire "verticale" l'extend dello splay [0,90]. Predefinito 10 gradi
  • Soglia vert.. Soglia della battuta verticale [0,90]. Predefinito 80 gradi
  • Extend manuale. Impostazione manuale "extend" (riferimento Sinistra/Destra). Predefinito NO
  • Chiusura anelli. Predefinito "Nessuna"
  • Pulsanti Prec./Suc.. Visualizzazione pulsanti Prec./Succ. nella finestra delle battute. Predefinito ON
  • Backsight. Campi backsight (battuta "a ritroso") nella finestra di inserimento battuta. Predefinito NO
  • Anomalia magnetica. Compensazione dell'anomalia magnetica. Predefinito NO
  • Timer direzioni. Secondi di attesa prima della misura. Predefinito 10s
  • Volume timer. Volume del segnale acustico del timer di misura direzione del tiro [10%,100%]. Predefinito 50%
OPZIONI POSIZIONE
  • Unità posizioni. gradi decimali {dec.deg} o gradi,minuti,secondi {ddmmss}. Predefinito ddmmss
  • CRS. Nome di CRS come in Proj4. Predefinito Long-Lat
OPZIONI ACCURATEZZA
  • Tolleranza accelerazione >0. Predefinito 400
  • Tolleranza campo magnetico >0. Predefinito 300
  • Tolleranza immersione magnetica >0. Predefinito 3 gradi

OPZIONI DISEGNO

  • Dialogo scelta prototipo. Selettore strumenti di disegno {recente, elenco, griglia, griglia tripla}. Predefinito recente
  • Numero di strumenti recenti {3, 4, 5, 6}. Predefinito 4
  • Side-drag. Abilita il trascinamento laterale nella finestra del disegno. Predefinito OFF
  • Regolazioni Zoom {disattivato, temporaneo, permanente}. Permanente predefinito
  • Controllo poligonale. Controlla se ci sono capisaldi non collegati alla poligonale. Predefinito ON
  • Controllo extend battuta. Controlla se gli extend di battuta sono impostati. Predefinito ON
  • Numero di backup. Predefinito 5
  • Intervallo di backup. Predefinito 60
    • Salva aree in xtherion- "stile". Predefinito NO
OPZIONI PUNTI
  • Punti icona senza zoom. Icone dei punti non vengono ingrandite. Predefinito NO
  • Scala simboli grafici. Predefinito 1.2
  • Dimensione scritte. Predefinito 24 px
OPZIONI LINEE
  • Spessore linee. Larghezza delle linee (N.B.: la linea di parete è il doppio). Predefinito 1 px
  • Stile della linea {Bezier, fine, normale, grossolana}. Predefinito normale
  • Spaziatura punti sulle linee. Spazio punto-punto, distanza minima tra i punti della linea. Predefinito 10 px
  • Direzione di sezione. "direzione tick" [1,20]. Predefinito 8 [x unità]
  • Punti di sezione. Aggiungi il punto della sezione Therion alla linea della sezione. Predefinito ON
  • Visibilità bordi area. Predefinito SI
  • Accuratezza curve Bezier. Predefinito 1
  • Angolatura Bezier. Soglia angolo linea di Bezier. Predefinito 20
OPZIONI DISEGNO
  • Spessore poligonale. Larghezza linea rilievo [0,5,10]. Predefinito 1 px
  • Dimensione nome capisaldi. Dimensione del nome del caposaldo. Predefinito 24 px
  • Raggio di punti selezione [0,5,100]. Predefinito 5 px
  • Raggio di selezione >=0. Predefinito 24
  • Raggio del cancellino. Dimensione della gomma. Predefinito 36
  • Sensibilità allo spostamento. Predefinito 60
  • Raggio di puntamento. Valore predefinito 16
  • Splay soglia verticale. Soglia di visualizzazione del piano di splay [0,90]. Predefinito 80 gradi
  • Soglia splay tratteggiati. Soglia di inclin. per il tratteggio degli splay [0,90]. Predefinito 50 gradi
  • Soglia sezione orizzontale [0,90]. Predefinito 70 gradi
AUTO-PARETI
  • Tipo di muri {nessuno, convesso}. Predefinito nessuno
  • Splay piano clino. Predefinito 70 gradi
  • Splay profilo clino. Predefinito 45 gradi
  • Separazione. Predefinito 0.1 m
  • Concavità. Predefinito 0.1
  • Passo. Predefinito 1 m

OPZIONI DISTOX

  • Bluetooth. Controlla se BT è abilitato all'avvio. Predefinito SI
  • Modalità dati di connessione {su richiesta, continua}. Predefinito su richiesta
  • Riconnessione automatica (in modalità continua). Predefinito NO
  • Tipo di socket BT {normale, non sicuro}. Predefinito normale
  • Dati modo Z6. Predefinito SI
  • Ritardo connessione >=0. Predefinito 2
  • Associazione automatica. Predefinito SI
  • Pausa dati[ms]. Predefinito 50
  • Ritardo di attesa dati pronti [ms]. Predefinito 100
  • Attesa dopo laser [ms]. Predefinito 1000
  • Attesa dopo tiro [ms]. Default 4000

OPZIONI DI CALIBRAZIONE

  • Politica gruppi {4x4 + M, Nx4, distanza). Predefinito Nx4
  • Tolleranza gruppi. Predefinito 40 [deg]
  • Errore. Errore algoritmo. Predefinito 0.000001
  • Iterazioni. Massimo cicli di iterazione. Predefinito 200
  • Dati grezzi. Mostra i dati grezzi di calibrazione. Predefinito NO
  • Algoritmo. Algoritmo di calibrazione {automatico, lineare, non lineare}. Auto predefinito
